# HTML syntax

```HTML = Hyper Text Markup Language```

## HTML tag, self-closing tag, attribute
```
<center> <!-- element -->
  <hr size="3" noshade> <!-- attribute -->
  <h1>Advantures</h1> <!-- opening and ending tags -->
  <br> <!-- self-closing tag-->
  <h3>by</h3>
  <br>
  <h2>Jagaa</h2>
  <hr size="3" noshade>
</center>
```
